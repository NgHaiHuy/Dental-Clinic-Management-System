## Dental Clinic Management System
Hệ thống Quản lý Phòng khám Nha khoa (Dental Clinic Management System) là giải pháp chuyển đổi số toàn diện nhằm tối ưu hóa toàn bộ quy trình vận hành và nâng cao chất lượng dịch vụ y tế. Hệ thống tích hợp các tính năng thông minh từ đặt lịch hẹn tự động, quản lý hồ sơ bệnh án điện tử trực quan với sơ đồ răng tương tác, đến kiểm soát chặt chẽ kho vật tư và tự động hóa báo cáo tài chính. Bằng cách kết nối mượt mà dữ liệu giữa chủ phòng khám, nha sĩ và bệnh nhân, dự án không chỉ loại bỏ các sai sót từ quy trình quản lý thủ công truyền thống mà còn giúp tối giản hóa thủ tục hành chính. Đây là công cụ cốt lõi giúp các phòng khám nha khoa nâng cao hiệu suất làm việc của đội ngũ y bác sĩ, tối ưu chi phí vận hành và mang lại trải nghiệm chăm sóc răng miệng hiện đại, chuyên nghiệp cho khách hàng.

## HƯỚNG DẪN TẠO NHÁNH (BRANCH) & ĐẨY CODE (PUSH) BẰNG GIT DESKTOP
### Bước 1: Tạo nhánh mới từ main
1. Mở **GitHub Desktop**, chuyển sang nhánh **`main`**.
2. Nhấn nút **Fetch origin** ở góc phải để cập nhật code mới nhất.
3. Nhấp vào mục **Current Branch** trên cùng $\rightarrow$ Chọn **New Branch**.
4. Đặt tên nhánh (Ví dụ: `feature-dang-nhap`, `fix-giao-dien`) $\rightarrow$ Nhấn **Create Branch**.
5. Nhấn nút **Publish branch** ở góc phải để đưa nhánh này lên hệ thống web.

### Bước 2: Làm việc và Đẩy code (Push) lên Git
1. Viết code/Sửa lỗi trên các phần mềm lập trình như bình thường.
2. Quay lại **GitHub Desktop**, bạn sẽ thấy danh sách các file đã thay đổi ở cột bên trái.
3. Nhìn xuống góc dưới cùng bên trái:
   * Ô **Summary (required)**: Gõ ngắn gọn việc bạn vừa làm (Ví dụ: *Sửa nút đăng nhập*).
   * Nhấn nút xanh **Commit to [tên-nhánh-của-bạn]**.
4. Nhấn nút **Push origin** ở góc trên cùng bên phải để đẩy code lên web.

### Bước 3: Tạo Pull Request để Trưởng nhóm duyệt
1. Sau khi Push xong, trên GitHub Desktop sẽ hiện nút **Create Pull Request**, hãy nhấn vào đó (hoặc lên thẳng trang web GitHub của dự án).
2. Nhấn nút xanh **Create pull request** để gửi yêu cầu gộp code vào `main`.
3. Nhắn Trưởng nhóm vào kiểm tra và bấm **Merge** (Gộp code). Không tự ý bấm Merge!
